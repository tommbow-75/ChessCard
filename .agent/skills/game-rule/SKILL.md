# ChessCard 遊戲規則指南

提供準確的開發協助時，務必參考核心遊戲機制。

## 核心工作流

1. 提議新卡牌或新規則時，核對基礎規則文件。
2. 嚴格依照定義使用謀略點數 (SP) 與士氣 (Morale) 數值。
3. 驗證卡牌效果時，確認其符合召喚卡或謀略卡的專屬框架。
